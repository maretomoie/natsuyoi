About the Event

きみ夏宵あそびとは

夏の夜——
花火が打ち上がり、縁日の賑わいが聞こえてくる、そんな特別な夜に
みんなで雀魂を囲む、3日間の夏祭り麻雀イベント。

参加者はリーダーペアやお祭り男・お祭り女と卓を囲み、
縁日ボーナスポイントを競い合います。

段位制限なし、対局数自由——
初心者も上級者も、自分のペースで楽しめる参加型フェスです。
